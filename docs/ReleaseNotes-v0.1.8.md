# BA Portal v0.1.8 – Resources

## Objective
Turn the placeholder Library area into a simple customer-friendly Resources centre.

## What changed
- Renamed Library to Resources.
- Added a proper Resources page.
- Added instant search.
- Added Recently Added, Training Guides, Project Documents and Useful Links sections.
- Added card-based resources with icons, descriptions, update text and action buttons.
- Kept document links as placeholders for now.

## What to test
1. Open `index.html`.
2. Check the sidebar says **Resources**, not Library.
3. Click Resources from the sidebar.
4. Click Resources from the dashboard card.
5. Try searching for:
   - proposal
   - chatgpt
   - checklist
   - nothing
6. Check the empty search message appears when nothing matches.
7. Check useful links open in a new tab.
8. Check the footer shows **v0.1.8 – Resources**.

## Known notes
- Project document cards currently use placeholder links.
- Real customer documents can be connected later by updating the resource data in `js/main.js`.
