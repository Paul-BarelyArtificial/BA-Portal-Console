# BA Portal Release Notes

## v0.1.9 – Dashboard Improvements
**Date:** 08 July 2026

### Objective
Make the dashboard feel more like a useful customer home page rather than a simple menu.

### Added
- New customer-facing dashboard hero area.
- Portal access summary card.
- Three simple status cards:
  - Project
  - Next Step
  - Support
- Clear **Start Here** section with priority actions.
- **What’s New** panel showing recent portal updates.
- Quick links row for fast navigation.
- More useful **My Account** page with access status placeholders.

### Changed
- Updated displayed version to **v0.1.9 – Dashboard Improvements**.
- Tidied old Library references in CSS after the v0.1.8 rename to Resources.

### Notes
- Account details are still static placeholders until authentication is connected.
- The dashboard is now ready to become dynamic later, using customer/project data.
