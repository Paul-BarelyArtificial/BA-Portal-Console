# BA Portal v0.2.1 – Firebase Authentication

## Objective
Connect the Customer Portal to the live Barely Artificial Firebase project and replace the demonstration access screen with real authentication.

## Included
- Email/password sign-in
- Remembered authenticated session
- Sign-out
- Protected portal shell
- Friendly authentication errors
- Signed-in email shown in My Account
- Decorative dashboard icons removed

## Test
1. Open the deployed GitHub Pages site.
2. Confirm the portal content is hidden behind the sign-in screen.
3. Try an incorrect password and confirm a clear error appears.
4. Sign in using the Firebase user account.
5. Refresh the page and confirm the session is remembered.
6. Open My Account and confirm the signed-in email is displayed.
7. Sign out and confirm the portal returns to the sign-in screen.
