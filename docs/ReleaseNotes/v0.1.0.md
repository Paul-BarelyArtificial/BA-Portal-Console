# BA Console v0.1.0 – Foundation

## Objective
Create the first version of the Barely Artificial Console as a separate application from the customer portal.

## What changed
- Added the core Console layout.
- Added navigation for Dashboard, Customers, Projects, Resources, Bookings, Reports and Settings.
- Added placeholder cards and tables to show intended future behaviour.
- Added versioning and release documentation.

## What to test
1. Open `index.html`.
2. Click every sidebar item.
3. Confirm each page changes correctly.
4. Check the layout on desktop and mobile width.
5. Confirm the footer shows `v0.1.0 – Foundation`.

## Known limitations
- No real login.
- No real customer data.
- No database connection.
- Buttons are placeholders unless they navigate inside the page.
