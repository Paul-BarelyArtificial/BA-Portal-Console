# BA Console v0.1.2 – Projects

## Objective
Add the Projects management area so the Console can represent the core relationship between customers, projects and resources.

## Added
- Projects table
- Project search
- Project status filters
- Sample project records
- Project detail panel
- Placeholder New Project dialog
- Updated dashboard project metric
