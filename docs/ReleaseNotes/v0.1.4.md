# BA Console v0.1.4 – Bookings

## Objective
Add booking management to the Console shell.

## Added
- Bookings management screen
- Booking search
- Upcoming / Completed / Cancelled filters
- Sample booking data
- Inline booking detail panel
- Close button on booking detail
- Placeholder + New Booking dialog
- Dashboard booking metric connected to sample data

## Notes
This release uses static sample data only. Booking records do not yet connect to Calendly or Firebase.
