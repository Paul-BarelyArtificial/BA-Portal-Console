# BA Console v0.2.0 – Firebase Authentication

## Objective
Protect the Barely Artificial Console with real Firebase authentication and an administrator role check.

## Included
- Email/password sign-in.
- Persistent sessions.
- Sign out.
- Firestore check against `admins/{uid}`.
- Access-denied screen for non-admin users.
- Clear sign-in and network error messages.

## Test
1. Open the deployed GitHub Pages site.
2. Confirm the Console is hidden behind the sign-in screen.
3. Try an incorrect password.
4. Sign in with the Firebase administrator account.
5. Refresh and confirm the session remains active.
6. Sign out and confirm the sign-in screen returns.
7. Optional: sign in with a Firebase user that has no matching admin document and confirm access is denied.

## Known limitation
Console data remains sample data. Firestore-backed customer records are planned for the next release.
