# BA Console v0.2.1 – Live Customers

## Objective
Connect the Customers screen to Firestore and allow administrators to create customer records.

## Test
1. Sign in as an administrator.
2. Open Customers. An empty state is expected before the first customer is created.
3. Click New Customer, complete the form and save.
4. Confirm the new customer appears without refreshing.
5. Refresh and confirm it remains.
6. Test search, filters, View and close.
