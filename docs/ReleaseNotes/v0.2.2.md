# BA Console v0.2.2 – Live Projects

## Objective
Replace sample projects with live Firestore projects connected to existing customers.

## Test
1. Sign in as an administrator.
2. Open Projects and confirm the empty or live list loads.
3. Create a project and select a live customer.
4. Confirm the project appears immediately and survives refresh.
5. Confirm the selected customer's project count increases.
6. Test search, status filters, View and close.
