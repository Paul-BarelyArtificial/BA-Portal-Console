# BA Console v0.2.3 – Live Resources

## Objective
Connect resource management to Firestore and Firebase Storage.

## Included
- Live resource list from Firestore
- Working New Resource form
- Customer dropdown populated from Firestore
- Project dropdown filtered to the selected customer
- File upload to Firebase Storage
- Useful Link support
- File size limit of 50 MB
- Live search, filters, detail view and dashboard count

## Test
1. Create a document or training guide with a small supported file.
2. Confirm upload progress appears and the resource is listed.
3. Refresh and confirm it remains.
4. Open the detail row and use Open file.
5. Create a Useful Link and confirm Visit link works.
6. Confirm the selected customer's project list is filtered correctly.
7. Confirm the project's resource count increases.
