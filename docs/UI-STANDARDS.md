# UI Standards

## Product names

- Barely Artificial Customer Portal
- Barely Artificial Admin Console

## Admin Console navigation

- Dashboard
- Customers
- Projects
- Library
- Bookings
- Reports
- Settings

## Language rules

- Say **Library item**, not resource.
- Say **All Customers**, **Selected Customers**, and **Internal**.
- Use short explanations beside unfamiliar choices.
- Avoid technical Firebase language in customer-facing screens.

## Experience rules

- A first-time user should understand where to go without training.
- Important actions must be visible and clearly named.
- Empty states must explain what to do next.
- Destructive actions require a clear warning or confirmation.
