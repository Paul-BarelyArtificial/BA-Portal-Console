# BA Admin Console v0.2.5 — Console Identity & Library Polish

## Purpose
Make the Admin Console visually distinct from the Customer Portal and improve the usability of Console forms.

## Changes
- Dark navy Console background, sidebar, cards and panels.
- Wider New Library Item dialog.
- Taller dropdowns and inputs with more internal padding.
- More vertical space between form fields.
- Clearer focus state on form controls.

## Firebase changes
None.
